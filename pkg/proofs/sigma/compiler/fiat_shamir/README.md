# Fiat-Shamir
This implements non-interactive compiler for any sigma protocol using Fiat-Shamir transform, see: [FS86].
(We use strong Fiat-Shamir which includes the Statement in the challenge and do hash chaining to unambiguously hash elements of the transcript)

[FS86]: <https://link.springer.com/content/pdf/10.1007/3-540-47721-7_12.pdf>
