# ZKPoK compiler
It's a compiler for constructing zero-knowledge proof of knowledge protocols from $\Sigma$-protocols.
For details see chapter 6.3 Proof of Knowledge of "Efficient Secure Two-Party Protocols" C. Hazay, Y. Lindell. [HL10]

[HL10]: <https://link.springer.com/book/10.1007/978-3-642-14303-8>
