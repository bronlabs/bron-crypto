# Generalized ElGamal Cryptosystem

This package implements the Generalized ElGamal cryptosystem over any finite cyclic group G,
following section 8.4.2 of "Handbook of Applied Cryptography" by A. Menezes et al.

## Reference

- section 8.4.2 of [https://cacr.uwaterloo.ca/hac/about/chap8.pdf](https://cacr.uwaterloo.ca/hac/about/chap8.pdf).
