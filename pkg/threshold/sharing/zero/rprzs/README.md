# `rprzs`: Replicated Pseudo-Random Zero Sharing

A protocol to generate replicated secret shares of zero among $n$ parties loosely based on [CDI05]. It requires a commitment scheme and a pseudo-random number generator. The setup is run once, and the sample can be run multiple times.

[CDI05]: https://iacr.org/archive/tcc2005/3378_342/3378_342.pdf
